- C: DynamoDBSaver は自動または context 管理でコネクションを扱います。
- D: Bedrock の呼び出し自体は synchronous に完了しています。
**関連知識:** Lambda 実行環境のライフサイクル

**Q19.** 〔カテゴリ: 概念理解〕
CI/CD 構成（`cicd-setup.yaml` 等）において、**OIDC (OpenID Connect)** を使用して AWS に接続する主なセキュリティ上のメリットは何か。

A. GitHub 上に AWS のアクセスキー ID やシークレットアクセスキーを永続的に保存しなくて済む
B. デプロイ速度が 10 倍以上高速化される
C. デプロイ後の Lambda 関数の処理効率が向上する
D. AWS マネジメントコンソールへのログインパスワードを GitHub と共通化できる

<details>
<summary>解答・解説を表示</summary>

**正解: A**
**なぜ正解か:** OIDC を使うと、GitHub Actions が動的に一時的な認証情報を AWS から取得するため、静的なシークレットを管理・流出させるリスクを最小化できます。
**不正解の理由:**
- B: 速度向上ではなく、安全性の向上が目的です。
- C: 実行時の効率には影響しません。
- D: アカウントパスワードの共有はセキュリティ上推奨されません。
**関連知識:** Identity Federation（ID フェデレーション）
**該当ファイル:** `cicd-setup.yaml` (プロジェクト構造より)

</details>

**Q20.** 〔カテゴリ: 機能間の関係〕
Lambda ハンドラー（`lambda_handler`）で `json.dumps(..., ensure_ascii=False)` を使用している理由として正しいものはどれか。

A. 数値データを文字列に変換するため
B. 日本語などのマルチバイト文字を、エスケープ（\uXXXX 形式）せずにそのまま出力するため
C. JSON データ全体のサイズを半分に圧縮するため
D. 外部のハッカーからの SQL インジェクションを防ぐため

<details>
<summary>解答・解説を表示</summary>

**正解: B**
**なぜ正解か:** 日本語メッセージが含まれる場合、`ensure_ascii=True`（デフォルト）だと文字化けのように見えるエスケープがなされるため、読みやすさを確保するために設定します。
**不正解の理由:**
- A: `json.dumps` 自体の基本機能であり、このオプション固有の効果ではありません。
- C: サイズ圧縮の効果はありません。
- D: セキュリティ対策（サニタイズ）ではなく、文字表現の変更です。
**関連知識:** UTF-8 エンコーディング
**該当ファイル:** `src/app.py`

</details>
