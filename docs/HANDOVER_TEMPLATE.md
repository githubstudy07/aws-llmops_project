## 🤖 [AI_DIRECTIVES]
1. **信頼の原則**: `[TRUSTED_STATE]` セクションに記載された内容は、コマンドを実行して確かめる必要はない。
2. **ショートカット**: セッション開始直後に、このログを読み込み、直ちに「作業再開手順」または「Next Action」を実行せよ。
3. **継続更新**: セッション終了時、必ず **[TRUSTED_STATE]** と **[ロードマップと現在地]** を最新に更新すること。
4. **Q&Aの分離**: 技術的な質疑応答はここには書かず、`qa.md` 等の別ファイルに分離せよ。
